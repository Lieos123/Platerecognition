## 项目简介

> 本项目是一个基于 **svm向量机分类识别** 的解决方案，旨在 **从视频中提取车牌并识别车牌号**，适用于 **中国内地车牌**。

> 独立项目，主要贡献者是Lieos123

## 功能特点

- 功能 1：提取关键帧序列
- 功能 2：sobel算法提取车牌
- 功能 3：车牌字符识别
